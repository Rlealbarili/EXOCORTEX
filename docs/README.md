# Official Documentation

This folder contains the official technical documentation for Exocortex Petrovich.

## Documents

- `ARCHITECTURE.md`: system design, runtime flow, and major components.
- `CONFIGURATION.md`: environment variables, secrets, and runtime tuning.
- `OPERATIONS.md`: service management, lifecycle commands, and runbook.
- `TROUBLESHOOTING.md`: incident diagnostics and known failure modes.
- `SECURITY.md`: security controls, secret handling, and guardrails.
- `DATABASE.md`: SQLite schema and state semantics.
- `DEVELOPMENT.md`: local development workflow and release checklist.

## Scope

This documentation covers the Petrovich project itself.
The `DOCS/` directory at repository root is Moltbook platform material and is not the project's official documentation.
